SPINResults Data Type

This class represents the results of the rule engine execution, consisting in the models and console logs generated by the rule engine library. Some of the results may be empty depending on the system call.

Properties
name data type description
model string Defined if the output is NOT in JSON format. The model union model (modified original model)
constraints string Defined if the output is NOT in JSON format. The model results for constraint checking
newTriplets string Defined if the output is NOT in JSON format. The new generated triples.
consoleLog string The results of the console log
jsonModel string If Json output specified: The model union model (modified original model)
jsonNewTriplets string If Json output specified: The new generated triples.
jsonConstraints string If Json output specified: The model results for constraint checking
